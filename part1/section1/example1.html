Variable & Data Types

  • JavaScript variables are containers for storing data values.
  • Javascript variable can hold many data types like number, string, array, object etc., but they are case sensitive.
  • var is used to create a variable.
  • JavaScript has dynamic types. The same variable can be used as different type.

Example 1

  • A variable declared without a value will have the value undefined.
  • When adding a number and a string, JavaScript will treat the number as a string.
  • JavaScript evaluates expressions from left to right.
  • If the first operand is a string, all operands are treated as strings.

Example 2

Example - var x; var x=5; var x = "Bootcamp"; var x = true;

Array Example- var bootcamp = ["HTML", "CSS", "JavaScript"];

Object Example- var topics = {html:"HTML & HTML 5", css:"CSS and SAAS", js:"JS Part 1 & Part 2"};

  • variable defined with no value has 'undefined' value.
  • Array indexes are zero-based.

Example 3

  • typeof operator is used to find the type of a JavaScript variable.
  • typeof variable without a value is undefined.
  • typeof null is object.
