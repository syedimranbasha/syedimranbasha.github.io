Object Prototypes

  • Create an object prototype using an object constructor function.
  • Create new objects from the same prototype using new keyword.
  • Add properties or methods to objects or object prototype

Example 1

Object prototype using an object constructor function:-- 

        function Bootcamp(session, count) {
            this.session = session;
            this.count = count;
        }

Create new objects from the same prototype (Bootcamp):-- 

        var day1 = new Bootcamp("HTML & CSS", "18");
        var day2 = new Bootcamp("SASS & CSS", "20");
  • The constructor function is the prototype for Bootcamp objects (day1 and day2).

Example 2

Add a new property and method to an existing object (day1):-- 

        day1.topicsCovered = 2;

        day1.myDetails = function () {
            console.log(this.session + "->" + this.count +"->" + this.topicsCovered);
        };
  • The property (topicsCovered) and method (myDetails) will be added to day1 only and not to day2.
  • See the console error for day2.

Example 3

Add a new property and method to existing prototype:-- 

        Bootcamp.prototype.duration = "2Hrs";
        Bootcamp.prototype.allDetails = function () {
            console.log(this.session + "->" + this.count + "->" + this.duration);
        };
  • The property (duration) and method (allDetails) are added to prototype.
  • And property and method ultimately added to day1 and day2 objects both.
