Inheritance

  • Inheritance between two classes; prototype is the way to achieve Inheritance in JavaScript.
  • Inherit using ChildClassName.prototype = new ParentClass();

Example 1

        var Person = function (name, age){
            this.name = name,
            this.age = age
        };

        Person.prototype.greet = function(){
            console.log('Hello, I\'m ' + this.name);
        }

        var Student = function(name,age){
            this.name = name,
            this.age = age
        };

        //Inheritance
        Student.prototype = new Person();  // Here's where the inheritance occurs
        Student.prototype.constructor=Student;  // Otherwise instances of Student would have a constructor of Person

        var PersonJohn = new Person("John", 22);
        var StudentMax = new Student("Max", 20);

        PersonJohn.greet(); // Hello, I'm John
        StudentMax.greet(); // Hello, I'm Max

        console.log (PersonJohn); //Person {name: "John", age: 22, greet: function}
        console.log(StudentMax); //Student {name: "Max", age: 20, greet: function}
  • Person is base class, Student is derived class.
  • Notice the __proto__ of Student and Person class.
