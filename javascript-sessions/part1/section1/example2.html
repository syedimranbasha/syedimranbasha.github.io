Variables Scope

  • JavaScript has function scope: the scope changes inside functions.
  • LOCAL Scope: Variables declared within a function, become local to the function.
  • GLOBAL Scope: Variables declared outside a function, becomes global.
  • Note: If you assign a value to a variable that has not been declared, it will automatically become a GLOBAL variable.
